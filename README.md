## Sobre o Projeto

Este projeto consiste em um **programa de consórcio desenvolvido na linguagem C**, com o objetivo de simular um sistema de gerenciamento de consórcios.

O sistema foi desenvolvido utilizando a linguagem **C** e possui um processo de compilação automatizado através de um **Makefile**, facilitando a execução do projeto.

## Tecnologias Utilizadas

* Linguagem C
* Makefile
* Compilador GCC

## Como Executar

Para compilar o projeto, certifique-se de possuir o GCC instalado.

No terminal, dentro da pasta do projeto, execute:

```bash
make
```

O comando irá utilizar o **Makefile** para realizar a compilação automática do programa.

Após a compilação, execute o arquivo gerado pelo processo de compilação.

## Funcionalidades

* Cadastro e gerenciamento de condutores
* Cadastro de Veiculos
* Processamento de Cota por idade , sigilos etc..
* Execução via terminal
